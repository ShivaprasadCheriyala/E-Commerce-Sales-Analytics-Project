# E-Commerce Sales Analytics Project

## 📌 Project Overview

This project was completed as part of the **Data Analytics Internship Program at Primeor Solutions**.

The objective of this project was to analyze E-Commerce sales data, generate business insights, and create an interactive dashboard to support data-driven decision-making.

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel
* SQL
* Power BI

---

## 📊 Dataset Information

* Records Analyzed: **51,000+**
* Domain: **E-Commerce Sales**
* Data Included:

  * Orders
  * Customers
  * Products
  * Sales
  * Profit
  * Discounts
  * Regions
  * Markets

---

## 📈 Project Tasks

### 1. Data Cleaning (Excel)

* Removed duplicate records
* Handled missing values
* Standardized data formats
* Corrected inconsistencies
* Validated numerical values

### 2. Excel Analysis

* Total Sales Analysis
* Total Profit Analysis
* Average Discount Analysis
* Top 5 Products by Sales
* Top 5 Loss-Making Products
* Region-wise Sales Analysis
* Category-wise Profit Analysis
* Segment-wise Sales Analysis
* Monthly Sales Trend Analysis

### 3. SQL Analysis

Performed business analysis using SQL queries:

* Top 10 Profitable Products
* Top 10 Customers by Sales
* Region-wise Sales Analysis
* Category-wise Average Profit
* Highest Discount Category
* Negative Profit Orders
* Monthly Sales Trend
* Market-wise Revenue Analysis
* Top Performing Sub-Categories
* Ship Mode Usage Analysis

### 4. Power BI Dashboard

Developed a 3-page interactive dashboard including:

#### Executive Overview

* Total Sales
* Total Profit
* Total Orders
* Average Discount

#### Sales Analysis

* Sales by Region
* Sales by Market
* Monthly Sales Trend
* Category Performance

#### Product & Customer Insights

* Top Products
* Top Customers
* Profit by Sub-Category
* Segment Analysis

---

## 💡 Key Business Insights

* Identified top-performing products and customer segments
* Analyzed regional and market performance
* Evaluated the impact of discounts on profitability
* Discovered profit trends and growth opportunities
* Generated actionable recommendations for business improvement
